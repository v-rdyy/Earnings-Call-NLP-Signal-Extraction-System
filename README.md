# Earnings Call NLP Signal Extraction System

## Project Overview
This project analyzes earnings call transcripts to predict short-term stock price direction using NLP features and machine learning.

## Project Structure
- `data/`: Raw and processed data
- `src/`: Source code modules
- `notebooks/`: Exploration notebooks
- `outputs/`: Saved models and results

## Setup Instructions
1. Create virtual environment: `python -m venv venv`
2. Activate virtual environment: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
3. Install dependencies: `pip install -r requirements.txt`
